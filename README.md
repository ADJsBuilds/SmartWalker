# Smart Walker Caregiver Dashboard MVP

Hackathon-ready caregiver / physiotherapist dashboard for Smart Walker seniors.

## What This Includes

- Next.js 14+ App Router app (in `frontend/`)
- Routes:
  - `/caregiver`
  - `/caregiver/alerts`
  - `/caregiver/residents/[id]`
  - `/caregiver/plans`
- Shared top-level caregiver tabs in `app/caregiver/layout.tsx`:
  - Residents / Alerts / Plans
- Supabase schema and seed:
  - `supabase/schema.sql`
  - `supabase/seed.sql`
- Event simulator:
  - `scripts/simulate_events.ts`

## Environment Variables

Set these in `frontend/.env.local`:

```bash
NEXT_PUBLIC_SUPABASE_URL=...
NEXT_PUBLIC_SUPABASE_ANON_KEY=...
SUPABASE_SERVICE_ROLE_KEY=...
```

## Setup

1. Install frontend dependencies:

```bash
cd frontend
npm install
```

2. Apply schema in Supabase SQL editor:
   - run `supabase/schema.sql`

3. Seed demo data:
   - run `supabase/seed.sql`

4. Start app:

```bash
cd frontend
npm run dev
```

Open: `http://localhost:3000/caregiver`

## Run Event Simulator

From repo root:

```bash
cd frontend
npm run simulate
```

This script inserts random events (`fall`, `near_fall`, `inactivity`, `heavy_lean`, `steps_update`) every ~2.5s.  
Alert rows are auto-generated by trigger logic in `schema.sql`.

## Demo Flow

1. Open `/caregiver` to view residents and statuses
2. Open `/caregiver/alerts` to triage alerts in realtime
3. Open a resident profile for charts, timeline, goals, notes, and check-ins
4. Open `/caregiver/plans` to create and assign plans
